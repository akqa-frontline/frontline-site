# Prism syntax highlighting

An extended version of the `@theme-ui/prism` renderer is available that adds line-highlighting and line numbers to the code blocks.

The component uses `prism-react-renderer` behind the scenes.

### Supported languages

The following languages are supported by `prism-react-renderer` by default:

- `markup`
- `bash`
- `clike`
- `c`
- `cpp`
- `css`
- `"css-extras"`
- `javascript`
- `jsx`
- `"js-extras"`
- `coffeescript`
- `diff`
- `git`
- `go`
- `graphql`
- `handlebars`
- `json`
- `less`
- `makefile`
- `markdown`
- `objectivec`
- `ocaml`
- `python`
- `reason`
- `sass`
- `scss`
- `sql`
- `stylus`
- `tsx`
- `typescript`
- `wasm`
- `yaml`

To add more languages, refer to the <ExternalLibraryLink title="prism-react-renderer FAQ" href="https://github.com/FormidableLabs/prism-react-renderer#faq"/>

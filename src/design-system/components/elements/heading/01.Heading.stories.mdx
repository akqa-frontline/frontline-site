# Heading

By default `<Heading/>` is rendered as a `h2` heading element to prevent SEO issues with too many `h1`.

You can set the heading level using the `as` prop.

  <Story name="Default">
    <Heading>Default</Heading>
  </Story>
</Preview>

# With Anchor

By adding an `id` prop to a `<Heading/>`, a child `<Link/>` will be rendered with a document fragment link.

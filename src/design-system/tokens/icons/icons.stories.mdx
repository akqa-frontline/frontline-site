# Feather Icons

These are all the available icons from the `feather-icons` icon set.

We encourage you use the icons via the `<Icon/>` component:

```jsx
import React from 'react';

import {ReactComponent as ActivityIcon} from '@/design-system/icons/source/activity.svg';
import Icon from '@/design-system/components/icon/Icon';

const SomeComponent = () => (
  <Icon>
    <ActivityIcon/>
  </Icon>
);
```

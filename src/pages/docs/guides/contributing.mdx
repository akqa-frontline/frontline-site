## Contributing to Frontline Docs

1. Become a member of the team. Ask Dennis to give you access to
https://github.com/akqa-frontline

2. Create a branch for you changes

3. Commit changes with `git commit` that uses commitizen to guide you to a fitting commit message

4. Create a pull request so the Frontline tream members can review your changes

## Contributing to Frontline

- This guide is found in the Frontline repo: [`CONTRIBUTING.md`](https://github.com/akqa-frontline/frontline/blob/master/CONTRIBUTING.md)
